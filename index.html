The Rumor With A Life Of Its Own

An interactive research experiment exploring how information changes when it travels through a human network.

Research Question

If the same message enters the same group of people, why can the ending be completely different?

This simulation models:

  • Social network structure
  • Personality differences
  • Memory errors
  • Information mutation
