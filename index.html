What to notice

Run it multiple times. The starting message is identical, but the ending can change because the path and the small random changes are different each time.

This model demonstrates how repeated transmission can create distortion. It does not represent real people perfectly: real rumors involve memory, emotions, incentives, culture, and many other factors.

The randomness comes from the rules of the simulation: each person has a chance to alter the message when passing it on.
